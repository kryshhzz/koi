

# koi : simple, small, quick web based editor in golang

## How to Download
Three steps to download koi

### 1. Download binary file 
[Linux](https://github.com/kryshhzz/koi/raw/refs/heads/main/download/koi-linux)
[Mac](https://github.com/kryshhzz/koi/raw/refs/heads/main/download/koi-max)
[Windows](https://github.com/kryshhzz/koi/raw/refs/heads/main/download/koi-windows.exe) 

### 2. Download `.KOI` folder
```
git clone github.com/kryshhzz/koi/.KOI/
```

### 3. move the `.KOI` folder to your home directory

#### for linux & mac
```
mv .KOI ~/
```

#### for windows
```
move .KOI %USERPROFILE%
```

## How to Run  
Go to your terminal where the binary file is located and run the respective command given below:
#### for linux
```
./koi-linux
```

#### for linux 
```
./koi-mac
```

#### for windows
```
koi-windows.exe
```
